# 🛒 MERN Stack E-Commerce Website

A complete **Full Stack MERN E-Commerce Website** built with modern web technologies. This project provides a seamless shopping experience with secure authentication, OTP verification, online payments, and a dedicated Admin Dashboard for managing the entire platform.

## 🚀 Features

### 👤 User Features

- User Registration & Login
- OTP Email Verification
- Secure Authentication & Authorization
- Dark / Light Mode
- Product Search and Filtering
- Product Details Page
- Add to Cart / Remove from Cart
- Live Cart Count Update
- Quantity Management
- Checkout System
- Stripe Payment Gateway Integration
- Order Placement & Order History
- Unique Order ID Generation
- Order Confirmation Messages
- User Profile Management
- Address Management
- Logout Functionality
- Pagination for Better User Experience
- Responsive Design for Mobile, Tablet, and Desktop

### 🛠️ Admin Features

- Separate Admin Panel

- Dashboard Analytics

- Revenue Analytics
- Charts and Data Visualization
